# Inno Store
 
Proof-of-Concept for a serverless Architecture in Azure with Monorepo and Infrastructure-As-Code

- Backend 
  - API: Azure Web App (Typescript, Node.js, GraphQL Server)
  - DB: Cosmos DB
- Frontend 
  - UI: Azure Web App (Typescript, Angular, Grapqhl Client)
- Infrastructure-As-Code
  - Bicep
  - Github Actions
